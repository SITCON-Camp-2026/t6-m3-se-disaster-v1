## Mermaid 流程圖

請讓 Codex 根據上面的自然語言描述產生 Mermaid。

請用 VS Code 預覽，確認流程圖能正常顯示。

```mermaid
flowchart TD
  RI[原始資訊進入] --> V{檢查來源與內容是否完整}
  V -->|不完整| NU[標示「需要人工確認」]
  V -->|完整| S{足以判斷是否可形成候選結果？}
  S -->|否| NP[標示「不能直接處理」]
  S -->|是| C[建立候選結果]
  NU --> LR[留下人工判斷紀錄]
  NP --> LR
  C --> HR[標示為「仍需人工確認」]
  HR --> LR
  C --> Q{是否可直接轉成任務？}
  Q -->|否| HR
  Q -->|是| T[建立候選任務（待確認）] --> LR
  subgraph Notes [註記]
    LR -.-> Record[紀錄：來源、判斷者、時間、理由]
  end
```
